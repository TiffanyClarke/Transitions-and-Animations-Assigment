Assignment 7: Transitions & Animations

Overview

In this assignment, you are to use transitions and/or animations to complete a given task or series of tasks for each activity.

Guidelines

The guidelines are simple:

  • Don't edit the HTML. You only need to open styles.css (not default.css) and start creating the CSS.
  • Read each activity task(s) extremely carefully to be sure you are doing exactly what is asked.
  • Read the intro comments at the top of the style.css - there is important information there!
  • When I say "the element", I am refering to the red box in each activity.
  • Unless otherwise specified, each transition should complete and then transition itself back to the default setting (no smooth sliding one direction, and jerking back to the original place!)
  • Please keep durations and delays short (nothing more than 5s) so I don't have to sit here for an hour waiting for something to move. Laammmeee!
  • You may (probably will) have to visit W3Schools to do some research!
  • Remember the property values for the original box (see them in the comments at the top of style.css). There is no need to re-declare those same values in your CSS and in fact, marks will be deducted for repeated values!
  • Hint: animation names don't take integers well (ie, "seven" works where "7" does not)
  • Double check each activity before submission to ensure it works! If it doesn't work when I grade it, it gets a zero.

Activity 1

Task: Make the element slide 100px to the right when hovered over.

Activity 2

Task: Make the element grow to twice it's original size when hovered over.

Activity 3

Task: Make the element shrink to half it's original size when hovered over.

Activity 4

Task: Over a two second period, make the element grow to 300px wide AND change colour to blue when hovered over. Give the change an easing value of ease-in-out.

Activity 5

Task: Using transforms, rotate the element a total of two full rotations over a three second period when hovered over. Hint: How many degrees is one full rotation?

Activity 6

Task: Using transforms, rotate the element counter-clockwise by one full rotation over a two second period using an easing value of linear. Also, make the effect happen one full second after the element is hovered over.

Activity 7

Task: Make the element hover like a cloud, 1em up then 1em back down to it's original position. Each iteration should take three seconds to complete and the effect should continue infinitely.

Activity 8

Task: Make the element become completely transparent over a three second period of time. The element should remain transparent when the effect is complete.
